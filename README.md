### Note

- 分享到朋友圈的办法，在电脑端打开，chrome 自带分享功能，在搜索框右边看到一个分享的 icon，点击后创建二维码
，使用微信扫码再分享到朋友圈即可，如果是在手机端直接分享，会缺少标题等等。
